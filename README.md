# dotfiles

My configuration files.

## Setup

Open a terminal and run following:

```bash
bash -c "$(curl -LSs https://raw.github.com/ymattu/rsetup/master/setup.sh)"
```

The setup list is:

- Download dotfiles
- Create symbolic links
- Install R, RStudio
- Install packages

## Warning
Sorry, this script is intended only for macOS.
